# 老照片修复工作台 · Argentum Photo Restoration Lab

**Argentum** 是一款基于纯前端（HTML5/Canvas/JavaScript）开发的高级老照片修复实验室。它针对老旧照片常见的模糊、噪点、对比度缺失、光照不均以及周期性网纹干扰，提供了一套从空间域（Spatial Domain）到频域（Frequency Domain）的完整数字图像处理（DIP）解决方案。

---

## 🚀 快速开始

本工具采用轻量化设计，无需安装 Python、Node.js 或任何后端依赖，只需浏览器即可运行：

1.  **保存代码**：将源代码保存为 `index.html`。
2.  **运行**：在 Chrome、Edge 或 Firefox 浏览器中直接打开该文件。
3.  **操作**：通过“点击上传”或“拖拽”载入照片，即可开始修复流程。

## 🛠 功能模块详解

### 1. 核心操作
* **📂 图像输入**：支持本地照片加载，自动初始化处理画布。
* **↺ 撤销与重置**：支持历史记录回溯，可随时撤销单步操作或一键恢复至原始图像。
* **⬇ 导出**：将修复后的高清结果以 PNG 格式保存至本地。
* **✨ 智能修复 (Auto Fix)**：集成化流水线，自动执行自适应均衡 (CLAHE)、Retinex 亮度校正与拉普拉斯锐化，适合快速出片。

### 2. 空间域增强 (Spatial Domain)
* **直方图均衡化 (HE)**：全局调整对比度，提升灰度覆盖范围。
* **自适应直方图均衡 (CLAHE)**：基于分块（Tile）的对比度受限增强，避免全局 HE 导致的局部过曝或噪声放大。
* **空域滤波**：
    * **中值滤波**：有效去除椒盐噪声，保持边缘清晰。
    * **高斯滤波**：平滑图像，去除高斯随机噪声。
* **锐化增强**：
    * **拉普拉斯锐化**：基于二阶微分算子提取并增强边缘。
    * **USM (Unsharp Masking)**：反锐化掩蔽，通过高斯模糊层与原图的差值进行细节补偿，效果比常规锐化更自然。

### 3. 频域处理 (Frequency Domain)
* **FFT 傅里叶变换**：将图像从像素空间转换到频率空间。
* **低通滤波器 (LPF)**：
    * **理想/高斯/巴特沃斯**：阻断高频噪声，实现平滑效果。
* **陷波去网纹 (Notch Filter)**：**本项目核心功能**。针对老照片常见的丝绸纹理、扫描网线，用户可点击频谱图中的干扰点，精准消除周期性噪声。

### 4. 物理/生理启发算法
* **Retinex 增强**：模拟人类视觉系统对物体的感知，消除光照不均，还原物体的本质反射颜色。
* **同态滤波 (Homomorphic)**：利用对数变换将照度分量和反射分量分离，在频域进行动态范围压缩和对比度增强。

## 🧠 算法原理说明

### 1. 空间域卷积 (Spatial Convolution)
空域操作通过 $3 \times 3$ 或 $5 \times 5$ 的模板（核）在图像上滑动。例如，**拉普拉斯算子**通过检测像素值的二阶突变来定位边缘。

### 2. 离散傅里叶变换 (DFT)
系统使用快速傅里叶算法（FFT）将图像数据映射到复数平面。
* **低频分量**：位于频谱中心，代表图像的整体色调和轮廓。
* **高频分量**：远离中心，代表细节、边缘及噪声。

### 3. 陷波处理逻辑
周期性噪声在频谱中表现为一对或多对远离中心的“亮点”。通过构建一个陷波滤波器 $H(u,v)$，在这些亮点处设置极低的增益，即可在不破坏主体细节的情况下彻底根除特定纹理。

## 📊 质量评估指标
系统实时计算并显示以下指标，帮助用户量化修复效果：
* **PSNR (峰值信噪比)**：评估修复后与原图的失真程度。
* **Entropy (信息熵)**：衡量图像携带细节信息的丰富程度。
* **Contrast (对比度)**：基于灰度标准差评估图像的视觉张力。
* **Sharpness (清晰度)**：利用梯度幅值评估边缘锐利度。
